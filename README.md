# HR Analytics & Workforce Dashboard (Power BI)

A sophisticated, multi-page Power BI dashboard designed to provide comprehensive insights into human resources data, including demographics, performance readiness, and active workforce status.

## [> Click here to View and Interact with the Dashboard (pbix file) <](https://github.com/aleksap430/Employee-Analytics-PowerBI-Dashboard/raw/refs/heads/main/HR.pbix)

*(Note: To view and interact with the full dashboard, you need to download and open the `.pbix` file in Power BI Desktop.)*

### Key Features

* **Interactive Multi-Page Navigation:** Custom-designed navigation bar for seamless movement between Overview, Demographics, and Performance/List sections.
* **Active vs. At-Risk Employee Tracking:** Clearly distinguishes between active employees, those eligible for retirement/redundancy ("Will be retrenched"), and candidates for promotion.
* **Professional UI/UX:** Clean design, standardized color palette, and clear visual hierarchy for business-ready reporting.

### Tech Stack

* **Data Preparation:** Power Query (ETL, Custom Columns, Conditional Columns).
* **Calculations:** DAX (Custom Measures for Attrition, Headcount, etc.).
* **Visualization:** Power BI Desktop.
* **Data Source:** (Opisno, npr. anonymized HR dataset with X rows).
